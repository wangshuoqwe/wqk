# wqk# 八皇后问题求解器（工程化实践）

本项目是《人工智能导论》课程作业01，使用AI辅助（Claude Code）完成八皇后问题的工程化实现。

## 实现思路
- 采用经典的回溯算法，递归尝试每一行放置皇后的位置。
- 使用列表存储已放置的皇后位置，索引为行号，值为列号。
- 通过检查列冲突和对角线冲突（行差与列差的绝对值相等）来判断安全性。
- `solve()` 方法返回所有解的列表（每个解为长度为N的整数列表）。
- 提供 `format_solution()` 方法将解转换为可读的棋盘字符串。
