# HW02：论文导读与 DeepSeek Chatbot 实践

## 任务一：论文导读
- **选题论文**：《我国海上风电发展现状与趋势综述》（严新荣等，《发电技术》2024年2月）
- **导读生成**：使用 DeepSeek 大模型，根据论文摘要及章节内容生成导读正文，并人工润色。
- **配图方式**：从论文原文中截取关键图表（表1、表3、图1~8、表5），在 Markdown 中手动插入，并添加图文注。图片存放于 `images/` 文件夹。
- **导读文档**：[导读_我国海上风电发展现状与趋势综述.md](导读_我国海上风电发展现状与趋势综述.md)

## 任务二：Chatbot 示例代码
- **API 平台**：火山引擎方舟平台（DeepSeek 模型）
- **运行环境**：Python 3.8+，依赖见 `requirements.txt`
- **配置说明**：
  1. 在火山引擎控制台获取 API Key 和 Bot ID。
  2. 在项目根目录创建 `.env` 文件，填入：
     ```
     ARK_API_KEY=your_key
     BOT_ID=your_bot_id
     ```
  3. 安装依赖：`pip install -r requirements.txt`
  4. 运行脚本：`python chatbot_deepseek.py`
- **示例输入/输出**：
  用户：介绍一下海上风电的漂浮式基础。
  deepseek：漂浮式基础适用于水深超过50米的深远海域，主要包括张力腿式、半潜式、立柱式和驳船式。我国已建成“三峡引领号”“海装扶摇号”等示范项目……
